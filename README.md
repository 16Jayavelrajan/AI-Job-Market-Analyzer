# AI Job Market Analyzer

AI Job Market Analyzer is an end-to-end AI/ML project designed to analyze job market trends and predict job experience levels using Natural Language Processing (NLP) and Machine Learning techniques. The system processes real-world job posting data, extracts relevant technical skills, performs text preprocessing, and applies TF-IDF vectorization with Logistic Regression for classification.

The project includes an interactive dashboard built using Streamlit, allowing users to visualize hiring trends, explore in-demand skills, and predict the experience level required for a job role based on job descriptions.

## Key Features

* Data cleaning and preprocessing using Pandas
* NLP-based text normalization and stopword removal
* Skill extraction from job descriptions
* TF-IDF feature engineering
* Machine Learning classification using Logistic Regression
* Interactive Streamlit dashboard
* Experience level prediction system
* Visualization of top skills and job trends

## Technologies Used

* Python
* Pandas
* Scikit-learn
* NLTK
* Streamlit
* Matplotlib

## Project Workflow

1. Collect and preprocess job posting data
2. Clean and normalize textual descriptions
3. Extract important technical skills
4. Convert text into numerical vectors using TF-IDF
5. Train a Machine Learning model for classification
6. Visualize insights through an interactive dashboard
7. Predict experience level from user-provided job descriptions

This project demonstrates practical applications of AI/ML, NLP, data preprocessing, and dashboard development in solving real-world analytics problems.
